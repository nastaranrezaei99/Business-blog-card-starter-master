## Overview

<img width="903" height="854" alt="image" src="https://github.com/user-attachments/assets/417fc2f0-09c9-4281-af29-09ced51886ed" />


This project is a responsive card component that matches the design from the devChallenges "Business Blog Card" challenge.  
It includes an image header with an SVG overlay, a title/description section, and an author profile area with a divider.

### What I learned

- How to position an SVG overlay on top of an image using `position: relative/absolute`.
- How to align elements cleanly with Flexbox (row and column layouts).
- The difference between `margin` (outside spacing) and `padding` (inside spacing) when building UI components.
- How to improve typography (font-family, font-weight, font-size, and spacing) to better match a given design.

### Useful resources

- [MDN - Flexbox](https://developer.mozilla.org/en-US/docs/Learn/CSS/CSS_layout/Flexbox) - Helped me understand alignment with `justify-content` and `align-items`.
- [MDN - position](https://developer.mozilla.org/en-US/docs/Web/CSS/position) - Useful for understanding absolute positioning for the SVG overlay.

### Built with

- Semantic HTML5 markup
- CSS (custom styles)
- Flexbox

## Features

This site was created as a submission to a [devChallenges](https://devchallenges.io/challenges-dashboard) challenge.

- Responsive card layout
